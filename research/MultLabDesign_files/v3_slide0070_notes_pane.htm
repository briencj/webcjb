The permutation tells us us that units 1,2 and 3,4 are interchanged and that units 3 and 4 are interchanged.
Instead of permuting the Blocks and Plots labels, we read the number in the permutation as specifying the unit that will get the associated treatment. For example, unit 4 - first number in permutation - will get treatment 1.  That is, this method of the permutation values is the same as permuting the Blocks and Plots only and then re-ordering all factors so that Blocks and Plots are in standard order.
The alternative interpretation , that I had been using, is that the value specifies the unit from which to get the treatment for the unit corresponding to the permutation value.  For example, the first value 4 specifies that the treatment for this unit is 4th in the list of treatments.  I believe that this is incorrect.
