Software and files

Make sure that R, RStudio, the two required packages, and their dependencies are installed. It is suggested that you use RStudio to install the packages that are on CRAN, because it will ensure that all the package dependencies are also installed.

R and RStudio

Firstly, if you have not already done so, R and then RStudio need to be downloaded and installed.

  1. R Version 4.2.2 - click here, download the binary appropriate to your machine and install.
  2. RStudio 2023.xxxx - click here, download the binary appropriate to your machine and install. RStudio is a set of integrated tools designed to help you be more productive with R. It includes a console, syntax-highlighting editor that supports direct code execution, as well as tools for plotting, history, debugging and workspace management.

Download and install dae from the CRAN repository

You need the Version 3.2.15 or later of dae. If you have installed an earlier version, it will need to be updated. The installed version can be checked using packageVersion("dae").

To install dae Version 3.2.15 from CRAN in RStudio, select Tools > Install Packages. Make sure that the Install from: option is Repository (CRAN, CRANextra). Enter dae and make sure that Install dependencies is checked. Click on Install.

Generally, the most recent version of dae is available for download from my rpackages repository.

Download and install odw

If you have not got a recent version of odw installed on your machine, then you will need an install for it. Once you have obtained an odw install, you can install it using RStudio by selecting Tools > Install Packages. Make sure that the Install from: option is Package Archive File(.zip, ;.tar.gz). Browse to where you saved the downloaded file, select the file and click on Install. Otherwise, use the install.packages function.

A Wheat data set

The Wheat data set from Gilmour et al. (1995) will be used in the practical sessions. It is available with the asremlPlus package from CRAN. If you have asremlPlus installed then it can be accessed from R by loading the library using library(asremlPlus) and then executing data(Wheat.dat).

Otherwise, you can download the file Wheat.dat.rda that contains the data.frame Wheat.dat. Right click on the link and ask to Save link as... The data.frame can then be loaded using load(file = "Wheat.dat.rda").

Manuals

Manuals for R packages are usually available in the doc subdirectory of the directory in which the package is installed. In a standard installation, this will be in the library subdirectory of the R installation directory. For dae, an overview of the package can be obtained using ?dae and a list of the available vignettes using vignette(package = "dae").
