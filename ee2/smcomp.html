Computing notes and files

R is the primary package that will be used to carry out the statistical procedures in this course. R is a free statiatical software package. Assistance with installing it is available at http://chris.brien.name/rpackages/. Also available there are some notes on using R and the dae package, that has a number of functions that are useful in the Design and ANOVA of experiments.

The following following R data files and scripts are available from this page:

  • CRDConcr.dat.rda
  • CRDDrug.dat.rda
  • CRDForm.dat.rda
  • Fac2ByPr.dat.rda
  • Fac2Prod.dat.rda
  • Frf5Fuel.dat.rda
  • Frf8Heli.2003.dat.rda
  • Frf8Heli.lay.rda
  • House.Prices.dat.rda
  • LSTurn.dat.rda
  • RCBDInse.dat.rda
  • Possum.dat.rda (from DAAG package)
  • Primates.dat.rda
  • Rocket.dat.rda
  • Rocket.r
