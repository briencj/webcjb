DAE library for S-Plus

The DAE library of S-Plus functions has been developed to facilitate the use of S-Plus for the Design and Analysis of Experiments.

Using it on a UniSA computer pool

If you want to access the files while using S-Plus in a UniSA pool, start the Windows Explorer and use Tools > Map Network Drive by putting \\cwpool0\briencj in the Folder box and clicking on the Finish button. Then, use library(DAE, lib.loc="z:") in S-Plus to attach the library -- note that, if the drive letter that the Folder is mapped to is not "z:", then you will have to replace "z:" with the drive letter used.

Obtaining it for use on a workstation

The DAE library can either be copied from the data share or downloaded from the web.

To copy it from the data share, you need to map a drive to \\cwpool0\briencj (as above). Then copy the entire DAE subdirectory onto a floppy disk. When you have inserted the floppy disk in the workstation, copy the directory to a suitable directory on your computer -- one good choice would be to put it in your default S-Plus workspace directory.

Alternatively, it can also be dowloaded from here in a self-extracting archive. First, you will need to create the DAE subdirectory in a suitable directory on your computer -- again a good choice would be to put it in your default S-Plus workspace directory. Then, download the self-extracting archive, saving it in DAE directory that you have created. Double-click on DAE.exe and the files will be extracted.
